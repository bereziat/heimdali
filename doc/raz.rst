.. _cmd-raz:

raz
====================================

**raz** Fill an image with zero values

Synopsis
------------------------------------

.. code-block:: bash

   raz  [-x <NX>] [-y <NY>] [-z <NZ>] [-i <IX>] [-j <IY>] [-k <IZ>] [--]
           [--version] [-h] <IMAGE> ...

Description
------------------------------------

**raz** fill the image `IMAGE` given in arguments with zero values.

+----------------+-----------------------------------------------------------+
| option         | description                                               |
+================+===========================================================+
| -x, -y, -z     | Size (counted from 0) of the subregion                    |
+----------------+-----------------------------------------------------------+

.. _format_options:

Format options
====================================

These are options describing the image format. There are used each time an
image is created.


+----------------+-----------------------------------------------------------+
| option         | description                                               |
+================+===========================================================+
| -z N           | Number of planes                                          |
+----------------+-----------------------------------------------------------+
| -y N           | Number of lines                                           |
+----------------+-----------------------------------------------------------+
| -x N           | Number of pixel per line                                  |
+----------------+-----------------------------------------------------------+
| -v N           | Number of value per pixel                                 |
+----------------+-----------------------------------------------------------+
| -r             | Floating point values                                     |
+----------------+-----------------------------------------------------------+

fzoom
====================================

**fzoom** enlarge or reduce an iamge.

Synopsis
------------------------------------

.. code-block:: bash

   fzoom  [--sc <X>] [--] [--version] [-h] <IMAGE-IN> <IMAGE-OUT>


Description
------------------------------------

*fzoom* increse or reduce the size of an image by lineary interpolation.
